# 流星雨动画页面总结

## 项目简介
一个基于 HTML5 Canvas 的**流星雨动画页面**，包含：
- 黑色星空背景
- 800 颗闪烁的“love”文字星星
- 20 条白色渐变流星
- 预留月亮位（代码已存在，图片未加载）

## 技术实现
- **HTML5 Canvas**：全屏画布，所有动画均绘制于 `<canvas id="stars">`
- **JavaScript**：
  - 面向对象封装 `Star` 与 `MeteorRain` 两类
  - 使用 `setTimeout` 递归实现帧循环（星星 100 ms、流星 2 ms）
  - 线性渐变与透明度变化实现流星拖尾与消失

## 核心对象
| 名称        | 职责                             | 关键属性/方法 |
|-------------|----------------------------------|---------------|
| `Star`      | 闪烁的“love”文字星星             | `text="love"`、`getColor()` 黑白切换 |
| `MeteorRain`| 带渐变的斜向下流星               | `angle=30°`、`alpha-=0.002`、`move()` |

## 视觉参数
- **背景**：纯黑 (`#000`)
- **星星**：文字“love”，黑白随机闪烁
- **流星**：
  - 长度 150–230 px，角度 30°
  - 渐变：白 → 随机灰 → 黑
  - 速度 1–2 px/帧，透明度递减

## 使用方法
1. 保存代码为 `流星雨.html`
2. 浏览器直接打开即全屏播放
3. 若想显示月亮，取消 `drawMoon()` 中 `moon.src` 注释并替换为真实图片地址

## 可定制项
| 变量/函数        | 含义               | 默认值 |
|------------------|--------------------|--------|
| `starCount`      | 星星数量           | 800    |
| `rainCount`      | 流星数量           | 20     |
| `MeteorRain.angle`| 流星倾斜角        | 30°    |
| `alpha -= 0.002` | 流星淡出速度       | 0.002  |
| `setTimeout` 间隔| 动画帧率           | 星星100 ms / 流星2 ms |

## 浏览器兼容性
- 支持 HTML5 Canvas 的现代浏览器均可正常运行
- 移动端可全屏自适应
