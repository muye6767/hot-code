# 生日快乐 总结

## 项目简介
- **单文件零依赖**的「生日快乐」祝福页，适合手机/电脑直接发送
- **视觉元素**：
  - 渐变梦幻背景 + 100 颗彩色 `bokeh` 气泡随机飘升
  - CSS 烟花（pyro）持续绽放
  - 奶油蛋糕 + 摇曳火焰蜡烛
  - 双层立体「生日快乐」文字 +  Emoji 🎉
- **移动端适配**：≤ 670 px 仅显示居中大字，节省性能

## 技术要点
| 技术 | 用途 |
|------|------|
| 纯 CSS 动画 | bokeh 上升、烟花爆炸、火焰摇曳、文字渐变 |
| CSS 变量/函数 | `radial-gradient`、`box-shadow` 绘制蛋糕层、烟花星 |
| 媒体查询 | 区分移动端与桌面端展示 |
| 零 JavaScript | 打开即播，无需额外资源 |

## 主要元素与类名
| 元素/类 | 说明 | 关键样式 |
|---------|------|----------|
| `bokeh` ×100 | 彩色气泡 | `border-radius:50%` + 随机 `transform` + `explosion` 动画 |
| `.pyro>.before/.after` | 两组烟花 | `box-shadow` 生成 50+ 彩色星点，`bang|gravity|position` 三动画叠加 |
| `.cake` | 蛋糕主体 | 多层 `box-shadow` 模拟奶油花边，共 20 层 |
| `.plate` | 底盘 | 金属光泽径向渐变 |
| `.candle + #flame` | 蜡烛火焰 | 线性渐变 + `flicker` 摇摆/放大动画 |
| `h1 / h2 / span` | 立体祝福 | 双层文字错位 + 渐变填色 + Emoji 装饰 |

## 可定制点
1. **文案**：直接修改 `<h1>生日快乐</h1>` 或 `<h2>生日快乐</h2>`
2. **主色调**：替换 `bokeh` 各子元素 `background-color`
3. **烟花颜色**：编辑 `.pyro` 的 `box-shadow` 色值列表
4. **蛋糕颜色**：调整 `.cake` 的 `box-shadow` 各层颜色
5. **动画时长/节奏**：修改 `animation-duration` / `animation-delay`

## 使用方法
1. 保存为 `生日快乐.html` → 双击打开即可全屏播放
2. 手机端自动切换简洁模式，仅显示居中大字
3. 可直接发送给好友/恋人，零依赖、无需联网

## 浏览器兼容
- 支持 CSS3 动画的现代浏览器均正常（Chrome、Edge、Firefox、Safari）
- 自带 `-webkit-` / `-moz-` / `-o-` / `-ms-` 前缀，兼顾旧内核
