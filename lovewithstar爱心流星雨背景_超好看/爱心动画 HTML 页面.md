爱心动画 HTML 页面总结

## 项目简介
一个基于 HTML5 Canvas 的爱心动画页面，包含：
- 闪烁的星星背景
- 流星划过效果
- 粒子爱心动画

## 技术实现
- **HTML5 Canvas**：用于绘制所有动画效果
- **JavaScript**：
  - 使用 `requestAnimationFrame` 实现流畅动画
  - 自定义 `Particle` 和 `ParticlePool` 类管理粒子
  - 数学函数绘制爱心形状

## 主要功能
### 1. 星星背景
- 随机生成 800 颗星星
- 星星颜色在灰色 (`#333`) 和粉色 (`#ea80b0`) 间随机切换
- 每 100 毫秒刷新一次，实现闪烁效果

### 2. 流星效果
- 20 颗流星随机出现
- 使用线性渐变实现流星拖尾
- 超出边界后自动重置位置

### 3. 爱心粒子动画
- 500 个粒子从爱心边缘发射
- 粒子速度、生命周期和大小可配置
- 使用心形参数方程：
x = 160 * sin³(t)
y = 130 * cos(t) - 50 * cos(2t) - 20 * cos(3t) - 10 * cos(4t) + 25

## 视觉设计
- **背景色**：黑色 (`#000`)
- **主色调**：粉色 (`#ea80b0`)
- **辅助色**：灰色 (`#333`) 和白色渐变

## 文件结构
  ```
love.html
└── 内联 CSS 样式
└── 内联 JavaScript 代码
```

## 使用说明
1. 将代码保存为 `love.html`
2. 用浏览器直接打开即可观看动画
3. 支持窗口大小自适应

## 自定义选项
在 `settings` 对象中可调整：
- `particles.length`：粒子数量 (默认 500)
- `particles.duration`：粒子生命周期 (默认 2 秒)
- `particles.velocity`：粒子速度 (默认 100 px/s)
- `particles.size`：粒子大小 (默认 20 px)
